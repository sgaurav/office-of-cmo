# Office of CMO — Ethos

The opinionated stance behind every skill in this pack.

## 1. Decisions over dashboards

Most marketing tooling adds surface area: more reports, more channels, more vanity metrics. We add a forcing function: **every skill ends in a decision document a human can read in five minutes**. If a skill produces a status update instead of a decision, it is broken.

## 2. One specialist per phase

Marketing is not undifferentiated "marketing work." A CMO sets strategy. A performance lead allocates budget. A creative director briefs creative. A trafficker traffics. We refuse to collapse these roles into a single mush prompt. Each skill is one role, doing what only that role should do.

## 3. Artifacts are the contract

Skills compose because the *outputs* of one phase are the *inputs* of the next, persisted as Markdown files at known paths. The next skill reads the prior artifact instead of re-asking the user. This is the difference between a workflow and a toolbox.

## 4. Quantify the kill criteria

Every strategy doc declares, in numbers, when a campaign dies. "Kill at >$60 CPA after $300 spend or 14 days, whichever first." Without quantified kill criteria, a program drifts forever. A skill that lets the user dodge this rule is failing them.

## 5. Channels need theses

No channel goes into the plan without a one-line thesis ("Senior Eng VPs cluster on LinkedIn for thought-leadership consumption"). "Let's also try TikTok" is not a thesis.

## 6. Message congruence is non-negotiable

Ad hook → landing page headline → CTA copy must repeat the same words. The `/ads-landing-brief` skill enforces this; reviewers fail any LP that violates it.

## 7. Memory beats restating

Once you know the ICP, the voice, the winning angles — those live in `cmo-memory/`. Skills read from memory by default. Re-asking the user the same questions across sessions is a tax we refuse to charge.

## 8. The CMO is opinionated

Below ~$50k/month spend there is roughly one right way to run paid acquisition: focused channel set, ruthless creative testing, fast kill cycles, single-objective campaigns. We hard-code that opinion. Power users can override; defaults are not neutral.

## 9. Talk like a marketer, not like AI

No "Here is a comprehensive overview of...". No bullet point soup. Skills produce documents that sound like a competent operator wrote them at 9am on Tuesday with a coffee.

## 10. Ship the workflow, then the breadth

Phase 1 ships ads end-to-end before Phase 2 ships content. A complete vertical is more valuable than a buffet of half-baked verticals. We resist the temptation to broaden until the current phase is dogfooded and tight.
