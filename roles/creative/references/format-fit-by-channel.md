# Format Fit by Channel

The format that wins depends on **where the audience is, in what mood, on what surface, with what attention**. This document maps audience-state × channel × format. Decisions that violate this matrix get rejected at brief.

## The audience-state taxonomy

The same person on Meta in the evening (zoning out) is a different audience than that same person searching on Google during work hours (problem-aware). Format fit follows the state, not the demographic.

| State | What they're doing | Attention budget | Right format characteristics |
|---|---|---|---|
| **Vertical-feed scroll** (Meta Reels, TikTok, Shorts) | Killing time, sound off, thumb-scrolling | 1.5 seconds per impression | Hook in first 1.5s, sound-off readable, native pacing |
| **In-feed scroll** (Meta feed, LinkedIn feed, X) | Mixed mode — some intent, mostly browsing | 3–5 seconds | Visual stop, layout that earns the dwell |
| **Searching** (Google, Bing, Reddit search) | Active problem, looking for answer | High intent, full attention | Copy-driven; visual is supporting (or absent in Search RSAs) |
| **Educational consumption** (LinkedIn feed for content, YouTube for tutorials, Reddit for deep threads) | Learning mode, willing to engage 2+ minutes | Long form OK | Document Ads, multi-page content, longer video |
| **Pre-roll / interstitial** (YouTube In-Stream, Spotify Audio) | Captive but adversarial — they want to skip | 5–6 seconds before skip | Earn the 5th second, brand callout by 3s |
| **Subreddit-native** (Reddit Promoted Post) | Community member, suspicious of ads | Will read if voice is right | Founder-POV, conversational, looks-like-a-post |
| **DM / inbox** (LinkedIn Message Ads, Conversation Ads) | Personal channel, high friction to ignore | 30 seconds reading time | Personal voice, single CTA, short |
| **Retargeting** (any channel, post-site-visit) | Some context, lower trust threshold | 2–3 seconds, recognition kicks in | Recall + new angle (don't repeat the prospecting ad) |

## Channel × audience state × format matrix

### Meta (Facebook + Instagram)

| Surface | Audience state | Best format | Worst format |
|---|---|---|---|
| Feed | In-feed scroll | 4:5 video (15–30s) or 4:5 image | 9:16 cut into 1:1 (forced format mismatch) |
| Reels | Vertical-feed scroll | 9:16 video, ≤15s, hook in 1.5s | Repurposed Meta-feed studio video |
| Stories | Vertical-feed scroll | 9:16 video or static, sticker-CTA | Long-form |
| Right column | Anything | (Don't run here — low quality) | All |
| Audience Network | Anything | (Default off; turn on after validation) | All |

### TikTok

| Audience state | Best format | Worst format |
|---|---|---|
| Vertical-feed scroll (FYP) | 9:16 native UGC video, 7–35s, hook in 1.5s, sound-on design | Studio-grade polished video; repurposed Meta creative |
| Spark Ads (boosted organic) | Authorized creator content, reuses existing engagement | Brand-feel "ad" content forced into Spark format |

TikTok rewards rawness. High-fidelity production tanks performance more often than not.

### LinkedIn

| Surface | Audience state | Best format | Worst format |
|---|---|---|---|
| Sponsored Content (feed) | Educational consumption / in-feed | Document Ad (4–6 pages) or Single Image with strong claim | 9:16 video forced into feed |
| Sponsored Video | In-feed | 16:9 or 1:1 video, ≤30s, hook in 3s, captions burned | TikTok-style native UGC (wrong register) |
| Conversation Ads | DM | Personal voice, ≤500 char message, 1–3 CTAs | Multi-step funnel attempts |
| Message Ads | DM | (Use only on warm audiences) | Cold message ads |
| Document Ads | Educational consumption | 4–6 page PDF, founder-led, lead-form on last page | <3 pages (no value to flip) or >8 pages (no one reads) |

LinkedIn Document Ads consistently outperform Single Image for content-led B2B. Default to Document for thought-leadership / Identity-angle briefs.

### YouTube

| Format | Audience state | Best for | Worst for |
|---|---|---|---|
| In-Stream skippable | Pre-roll | 30s with hook in 5s, brand callout by 3s | <15s (no setup) or >60s (loses too many) |
| Bumper (6s non-skippable) | Pre-roll | Awareness reinforcement; pair with In-Stream campaigns | Direct response (too short to convert) |
| In-Feed (formerly Discovery) | Educational consumption | 30–90s tutorial / demo / "how it works" | <15s (won't earn the click) |
| Shorts | Vertical-feed scroll | Same energy as TikTok; reuse 9:16 native creative | Studio-grade polished |

### Google Search

| Audience state | Best format | Worst format |
|---|---|---|
| Searching (high intent) | RSA copy + sitelinks + callouts; LP carries the weight | Visual creative (not where Search inventory lives) |
| Searching (PMax) | Asset group with photos + headlines + descriptions | Same assets as Display campaigns (PMax expects diverse) |
| YouTube via Google Ads | See YouTube section above | — |

Search is copy-driven, not visual-driven. The brief's "format spec" for a Search campaign is essentially the LP plus the RSA copy structure.

### Reddit

| Surface | Audience state | Best format | Worst format |
|---|---|---|---|
| Promoted Post (in-feed) | Subreddit-native | Founder-POV text post, conversational, ≤300 char title, body that earns the read | Polished image ad with stock-photo-style imagery |
| Conversation Ads | Subreddit-native | Interactive, branching CTAs, poll-style hooks | Linear "click here to learn more" |
| Image / Video ads | Subreddit-native | If used, must look unstaged — UGC-feel, "real" subject | Studio-feel; reads as ad, gets ignored |

Reddit's audience is allergic to marketing voice. Even visual ads need to look unstaged.

### X (Twitter)

| Surface | Audience state | Best format | Worst format |
|---|---|---|---|
| Promoted Tweet | In-feed | Boost an organic tweet that's already performing | Promoted tweet from cold creative with no organic context |
| Image / Video | In-feed | 4:5 mobile-optimized; founder-POV text + image | Long-form carousel (X audience won't dwell) |
| Take-Over (Trend / Timeline) | Captive | Premium launch moments only; pair with coordinated organic | Solo Take-Over without organic activity |

Promoted Tweets that boost organic posts consistently outperform cold creative on X.

### Bing / Microsoft

| Audience state | Best format | Worst format |
|---|---|---|
| Searching | RSA copy (basically same as Google) | Visual creative |
| Audience Network | (Use cautiously — quality varies) | Same as Display |

Bing copy can usually be imported from Google with minimal changes; the format-fit is identical.

## Decisions that violate the matrix

When a brief proposes a format that violates this matrix, reject before shipping:

- **9:16 video repurposed to Meta feed without 4:5 cut** → mobile feed plays at wrong aspect; tanks
- **Studio-grade polished video on TikTok or Reddit** → reads as ad; the audience scrolls past
- **Single Image on LinkedIn for a content-led brief** → Document Ads outperform; switching format alone often improves CTR 30%+
- **<15s hook-only video for YouTube In-Stream** → no setup; viewer skips before the message lands
- **TikTok-style native UGC on LinkedIn Sponsored Video** → wrong register; B2B audience reads as unprofessional
- **Polished image ad on Reddit** → community sniffs it out; comments tank the ad
- **9:16 video forced into Google Search** → channel doesn't have visual inventory at scale; spend goes to YouTube placements you may not have intended

## Production tier ↔ channel matrix

What production tier wins depends on the channel:

| Channel | Production tier that wins | Tier that tanks |
|---|---|---|
| TikTok | iPhone-grade UGC | Studio-grade |
| Meta Reels | iPhone-grade UGC OR studio-grade (both work) | Slack between (looks like a failed UGC attempt) |
| Meta feed | Studio-grade OR clean UGC | Phone-shot in poor lighting |
| LinkedIn | Studio-grade or clean studio illustration | Selfie-style UGC (wrong register) |
| YouTube In-Stream | Studio-grade preferred; UGC works for testimonial | Bad audio (especially) |
| Reddit | UGC / unstaged photo / typographic | Studio-grade (reads as ad) |
| X | Founder-POV / native / typographic | Studio-grade |
| Google Search | N/A (copy-driven) | — |
| Bing | N/A (copy-driven) | — |

Production-tier mismatches are visible to audiences within 1.5 seconds. Don't brief studio-grade for TikTok; don't brief UGC for LinkedIn Sponsored Video.

## When the user proposes a format outside the matrix

Two cases:

1. **They want to test something the matrix says is wrong** — push back: "Here's why this format usually loses on this channel; here's the format that usually wins. If you want to test the unusual format anyway, that's a deliberate experiment with explicit risk; let's frame it that way and define what would tell us we're right or wrong."

2. **The matrix doesn't cover their case** (e.g., a niche channel I haven't documented) — work from first principles: what's the audience's state on this surface? What's the attention budget? What format respects both? Document the new line in the matrix afterward.

## Quick decision shortcut

If you're in a hurry and need a format-fit gut check:

```
1. Is the audience scrolling vertically? → 9:16 video, hook in 1.5s, sound-off
2. Is the audience searching? → copy-driven RSA + good LP
3. Is the audience consuming content? → Document Ad / long-form video / multi-page
4. Is the audience captive (pre-roll)? → 30s with hook in 5s, brand by 3s
5. Is the audience in DM? → personal voice, ≤500 char
6. Otherwise → in-feed standard (4:5 image or video)
```

Most format-fit decisions land in #1, #3, or #6.
